# 入門｜形式言語・オートマトン・チューリングマシン

## What's this?

- プログラマとしての能力を向上させるために、アーキテクチャを設計する力が必要になった
- そのために、コンピュータとは何か、計算とは何か理解する必要性を感じた
- よって、情報科学・認知科学の重要な基礎理論である形式言語・オートマトン理論について学ぶものである
